01
Instalación — CDN, npm y Vite con Laravel
CDN · npm · Vite · viewport meta · estructura base
▼

Bootstrap 5.3 se puede usar de tres maneras según el contexto del proyecto. La más rápida para aprender es el CDN; la correcta para producción con Laravel es Vite.

Opción A — CDN (rápido, sin instalación)

HTML — Plantilla base con CDN
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <!-- Viewport obligatorio para responsive -->
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Mi página</title>

  <!-- Bootstrap 5.3 CSS -->
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css"
        rel="stylesheet">
</head>
<body>

  <h1>¡Hola Bootstrap!</h1>

  <!-- Bootstrap 5.3 JS (con Popper incluido) -->
  <!-- Siempre AL FINAL del body, antes de </body> -->
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>

Opción B — npm (proyectos sin Laravel)

Terminal
# Instalar Bootstrap con npm
npm install bootstrap@5.3.3

# Importar en tu archivo JS principal
# import 'bootstrap/dist/css/bootstrap.min.css';
# import 'bootstrap/dist/js/bootstrap.bundle.min.js';

Opción C — Vite con Laravel (recomendado en el curso)

Terminal — Laravel + Bootstrap
# Instalar Bootstrap y dependencias
npm install bootstrap @popperjs/core

# En resources/js/app.js — agregar:
# import 'bootstrap';

# En resources/sass/app.scss — agregar:
# @import 'bootstrap/scss/bootstrap';

# Compilar
npm run dev
⚠️
El <meta name="viewport"> es obligatorio. Sin él, Bootstrap no funciona correctamente en móviles aunque el CSS esté cargado.
🆕
Bootstrap 5.3 eliminó jQuery como dependencia. No necesitas incluirlo. El JS de Bootstrap es puro vanilla JavaScript con Popper.js incluido en el bundle.
02
Sistema de Grid — 12 columnas responsive
container · row · col · breakpoints · offset · order
▼

El grid de Bootstrap usa flexbox y se basa en 12 columnas. Siempre necesitas la jerarquía: container → row → col.

Breakpoints oficiales de Bootstrap 5.3

BreakpointClase prefijoAncho mínimoContainer máx.
Extra smallcol-< 576px100%
Smallcol-sm-≥ 576px540px
Mediumcol-md-≥ 768px720px
Largecol-lg-≥ 992px960px
Extra largecol-xl-≥ 1200px1140px
XXLcol-xxl-≥ 1400px1320px

Ejemplos de grid

HTML — Grid básico
<!-- Container: centra el contenido con padding lateral -->
<div class="container">

  <!-- Row: agrupa columnas, aplica gutter negativo -->
  <div class="row">

    <!-- 3 columnas iguales (4+4+4 = 12) -->
    <div class="col-md-4">Columna 1</div>
    <div class="col-md-4">Columna 2</div>
    <div class="col-md-4">Columna 3</div>
  </div>

  <!-- Columnas responsive: full en móvil, mitad en tablet, tercio en desktop -->
  <div class="row">
    <div class="col-12 col-sm-6 col-lg-4">Responsive</div>
    <div class="col-12 col-sm-6 col-lg-4">Responsive</div>
    <div class="col-12 col-sm-6 col-lg-4">Responsive</div>
  </div>

  <!-- Auto-layout: columnas de ancho igual -->
  <div class="row">
    <div class="col">Auto</div>
    <div class="col">Auto</div>
    <div class="col">Auto</div>
  </div>

  <!-- Offset: desplazar una columna a la derecha -->
  <div class="row">
    <div class="col-md-4 offset-md-4">Centrado</div>
  </div>

  <!-- Gutters: espacio entre columnas -->
  <div class="row g-3">       <!-- g-0 a g-5 -->
    <div class="col-6">Con gutter</div>
    <div class="col-6">Con gutter</div>
  </div>

</div>

JavaScript de Bootstrap
data-bs-* · API JS · eventos · Modal.getInstance · Toast
▼

Bootstrap 5.3 expone una API JavaScript para controlar los componentes desde código, sin depender solo de los atributos data-bs-*.

Dos maneras de controlar componentes

JS — API de Bootstrap
// ─── FORMA 1: Atributos data-bs (sin JS) ─────────────
// Solo HTML, Bootstrap hace el resto automáticamente
// data-bs-toggle="modal" data-bs-target="#id"

// ─── FORMA 2: API JavaScript ─────────────────────────

// Modal — abrir/cerrar desde JS
const modalEl = document.getElementById('miModal');
const modal = new bootstrap.Modal(modalEl);
modal.show();
modal.hide();

// Obtener instancia existente
const inst = bootstrap.Modal.getInstance(modalEl);

// Toast — notificación temporal
const toastEl = document.getElementById('miToast');
const toast = new bootstrap.Toast(toastEl, {
  delay: 3000,       // milisegundos
  autohide: true
});
toast.show();

// Tooltip — inicializar todos los tooltips
document.querySelectorAll('[data-bs-toggle="tooltip"]')
  .forEach(el => new bootstrap.Tooltip(el));

// Escuchar eventos de Bootstrap
modalEl.addEventListener('shown.bs.modal', () => {
  console.log('Modal abierto');
});

modalEl.addEventListener('hidden.bs.modal', () => {
  console.log('Modal cerrado');
});

Toast — notificación no intrusiva

HTML + JS — Toast
<!-- Contenedor posicionado -->
<div class="toast-container position-fixed bottom-0 end-0 p-3">
  <div id="miToast" class="toast align-items-center text-bg-success">
    <div class="d-flex">
      <div class="toast-body">
        ✅ Guardado correctamente.
      </div>
      <button class="btn-close btn-close-white me-2 m-auto"
              data-bs-dismiss="toast"></button>
    </div>
  </div>
</div>

<!-- Mostrar el toast desde JS -->
<script>
  function mostrarToast() {
    const t = new bootstrap.Toast(
      document.getElementById('miToast')
    );
    t.show();
  }
</script>
✅
Integración con Laravel Blade: muestra un Toast automáticamente si hay un mensaje flash de sesión: @if(session('success')) <script>mostrarToast()</script> @endif

Personalización — Variables CSS y Sass
--bs-primary · data-bs-theme · override Sass · color-mode
▼

Bootstrap 5.3 usa variables CSS nativas que puedes sobreescribir sin tocar el código fuente. Para personalizaciones profundas, usa Sass.

Opción A — Variables CSS (rápida, sin build)

CSS — Sobreescribir variables Bootstrap
/* En tu CSS después del CDN de Bootstrap */
:root {
  /* Cambiar el color primario */
  --bs-primary:         #7952b3;
  --bs-primary-rgb:     121, 82, 179;

  /* Cambiar fuente principal */
  --bs-font-sans-serif: 'Inter', sans-serif;

  /* Cambiar border radius global */
  --bs-border-radius:   8px;
  --bs-border-radius-lg: 12px;

  /* Cambiar sombras */
  --bs-box-shadow:      0 4px 20px rgba(0, 0, 0, 0.08);

  /* Cambiar colores de fondo */
  --bs-body-bg:         #f8f5f0;
  --bs-body-color:      #1a1218;
}

Dark Mode nativo — Bootstrap 5.3

HTML + JS — Color modes
<!-- Activar modo oscuro globalmente -->
<html lang="es" data-bs-theme="dark">

<!-- Solo en una sección -->
<div data-bs-theme="dark">
  <!-- Todo dentro de aquí usa tema oscuro -->
</div>

<!-- Toggle desde JS -->
<button onclick="toggleTheme()" class="btn btn-secondary">🌙 Modo oscuro</button>

<script>
  function toggleTheme() {
    const html = document.documentElement;
    const current = html.getAttribute('data-bs-theme');
    html.setAttribute('data-bs-theme', current === 'dark' ? 'light' : 'dark');
  }
</script>

SCSS — resources/sass/app.scss
