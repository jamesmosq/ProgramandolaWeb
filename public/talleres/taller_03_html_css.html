01
Estructura HTML5 semántica
DOCTYPE · head · body · header · main · nav · footer · section · article
▼

HTML5 introduce etiquetas semánticas que describen el significado del contenido, no solo su apariencia. Esto mejora SEO, accesibilidad y legibilidad del código.

📁
Crea una carpeta practica_html/. Dentro crea index.html. Ábrelo en el navegador con Live Server de VS Code o simplemente abriendo el archivo.
HTML
<!-- Estructura base de EduCode -->
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>EduCode — Aprende programación</title>
  <meta name="description" content="Plataforma educativa EduCode">
</head>
<body>

  <header>
    <nav>
      <a href="/">EduCode</a>
      <ul>
        <li><a href="/modulos">Módulos</a></li>
        <li><a href="/login">Ingresar</a></li>
      </ul>
    </nav>
  </header>

  <main>
    <section class="hero">
      <h1>Aprende programación paso a paso</h1>
      <p>Desde SQL hasta Laravel. Tú marcas el ritmo.</p>
      <a href="/register">Comenzar gratis</a>
    </section>

    <section class="modulos" aria-label="Lista de módulos">
      <h2>Ruta de aprendizaje</h2>
      <article class="modulo-card">
        <span class="icono">🗄️</span>
        <h3>Bases de Datos</h3>
        <p>SQL, MySQL y normalización</p>
      </article>
    </section>
  </main>

  <footer>
    <p>© 2024 EduCode</p>
  </footer>

</body>
</html>

02
Tailwind CSS — primeros pasos
CDN · utilidades · bg- · text- · p- · m- · rounded · shadow
▼

Tailwind CSS es un framework de utilidades: en vez de escribir CSS custom, aplicas clases pequeñas directamente en el HTML. Es lo que usa EduCode en producción.

🚀
Para practicar sin Node.js, usa el CDN de Tailwind. En producción (Laravel) se instala via npm. Agrega esto en el <head> de tu archivo de práctica.
HTML
<!-- Agrega en el <head> para practicar -->
<script src="https://cdn.tailwindcss.com"></script>
<script>
  tailwind.config = {
    darkMode: 'class',
    theme: {
      extend: {
        colors: {
          brand: '#00d4ff',
        }
      }
    }
  }
</script>

<!-- Clases básicas de Tailwind -->
<body class="bg-gray-950 text-gray-100 min-h-screen">

  <!-- Espaciado: p-4 = padding 1rem, m-2 = margin 0.5rem -->
  <div class="p-6 m-4 bg-gray-900 rounded-2xl shadow-xl">

    <!-- Tipografía -->
    <h1 class="text-3xl font-bold text-cyan-400 mb-2">
      EduCode
    </h1>

    <p class="text-gray-400 text-sm leading-relaxed">
      Aprende programación desde cero
    </p>

    <!-- Botón -->
    <a href="#" class="inline-block mt-4 px-6 py-2 bg-cyan-500 hover:bg-cyan-400
      text-white font-semibold rounded-xl transition-colors">
      Comenzar
    </a>

  </div>

</body>

Referencia rápida de clases

  • bg-gray-900 — fondo gris oscuro. bg-cyan-500 — fondo cyan
  • text-xl — texto grande. font-bold — negrita
  • p-4 — padding 1rem. px-6 — padding horizontal 1.5rem
  • rounded-xl — bordes redondeados. shadow-lg — sombra
  • hover:bg-cyan-400 — cambia color al hacer hover

03
Flexbox — layout lineal
flex · items-center · justify-between · gap · flex-wrap
▼

Flexbox organiza elementos en una dimensión (fila o columna). Es perfecto para navbars, headers y filas de elementos. En Tailwind se activa con la clase flex.

HTML
<!-- Navbar de EduCode con Flexbox -->
<header class="bg-gray-900 border-b border-gray-800">
  <nav class="max-w-5xl mx-auto px-6 py-4
    flex items-center justify-between">

    <!-- Logo -->
    <a href="/" class="text-xl font-bold text-cyan-400">
      ⚡ EduCode
    </a>

    <!-- Links del centro (ocultos en móvil) -->
    <ul class="hidden md:flex items-center gap-6 list-none">
      <li><a href="#" class="text-gray-400 hover:text-white text-sm transition">Módulos</a></li>
      <li><a href="#" class="text-gray-400 hover:text-white text-sm transition">Mi progreso</a></li>
    </ul>

    <!-- Botones de acción -->
    <div class="flex items-center gap-3">
      <a href="#" class="text-sm text-gray-400 hover:text-white">Ingresar</a>
      <a href="#" class="px-4 py-2 bg-cyan-500 hover:bg-cyan-400
        text-white text-sm font-semibold rounded-xl transition">
        Registrarse
      </a>
    </div>
  </nav>
</header>

04
CSS Grid — cuadrícula de módulos
grid · grid-cols · gap · auto-fill · minmax
▼

Grid organiza elementos en dos dimensiones (filas y columnas). Es ideal para la cuadrícula de tarjetas de módulos. Con auto-fill y minmax es responsive automáticamente.

HTML
<!-- Cuadrícula de módulos -->
<section class="max-w-5xl mx-auto px-6 py-16">
  <h2 class="text-2xl font-bold text-center mb-10">
    Ruta de aprendizaje
  </h2>

  <!-- Grid: 1 col en móvil, 2 en tablet, 3 en desktop -->
  <div class="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-6">

    <!-- Tarjeta 1 -->
    <article class="bg-gray-900 border border-gray-800 rounded-2xl p-6
      hover:border-cyan-500/50 transition-colors group cursor-pointer">
      <span class="text-4xl">🗄️</span>
      <h3 class="mt-4 font-bold text-lg group-hover:text-cyan-400 transition">
        Bases de Datos
      </h3>
      <p class="text-gray-400 text-sm mt-2">
        SQL, MySQL y relaciones
      </p>
      <div class="mt-4 flex items-center justify-between text-xs text-gray-500">
        <span>4 lecciones</span>
        <span class="text-cyan-400">Ver →</span>
      </div>
    </article>

    <!-- Repetir para los otros 4 módulos... -->
  </div>
</section>

Formulario de login estilizado
form · input · label · button · focus: · ring
▼

El formulario de login es la primera pantalla que ve el estudiante. Debe ser limpio, accesible y con buen feedback visual al hacer focus en los campos.

HTML
<div class="min-h-screen bg-gray-950 flex items-center justify-center px-4">
  <div class="w-full max-w-md">

    <!-- Header del form -->
    <div class="text-center mb-8">
      <div class="text-4xl mb-3">⚡</div>
      <h1 class="text-2xl font-bold">Iniciar sesión</h1>
      <p class="text-gray-400 text-sm mt-1">Bienvenido de nuevo a EduCode</p>
    </div>

    <!-- Formulario -->
    <form class="bg-gray-900 border border-gray-800 rounded-2xl p-8 space-y-5">

      <!-- Campo email -->
      <div>
        <label for="email" class="block text-sm font-medium text-gray-300 mb-1.5">
          Correo electrónico
        </label>
        <input type="email" id="email" name="email"
          placeholder="tu@correo.com" required
          class="w-full bg-gray-800 border border-gray-700 rounded-xl
            px-4 py-3 text-sm text-gray-100 placeholder-gray-500
            focus:outline-none focus:ring-2 focus:ring-cyan-500 focus:border-transparent
            transition">
      </div>

      <!-- Campo password -->
      <div>
        <label for="password" class="block text-sm font-medium text-gray-300 mb-1.5">
          Contraseña
        </label>
        <input type="password" id="password" name="password"
          placeholder="••••••••" required
          class="w-full bg-gray-800 border border-gray-700 rounded-xl
            px-4 py-3 text-sm text-gray-100 placeholder-gray-500
            focus:outline-none focus:ring-2 focus:ring-cyan-500 focus:border-transparent
            transition">
      </div>

      <!-- Botón submit -->
      <button type="submit"
        class="w-full py-3 bg-cyan-500 hover:bg-cyan-400
          text-white font-semibold rounded-xl transition-colors">
        Ingresar
      </button>

    </form>
  </div>
</div>

07
Diseño responsive completo
sm: · md: · lg: · hidden · block · mobile-first
▼

Tailwind sigue la filosofía mobile-first: primero diseñas para móvil y luego añades clases para pantallas más grandes con los prefijos sm:, md:, lg:.

HTML
<!-- Breakpoints de Tailwind:
  sm:  640px  (tablets pequeñas)
  md:  768px  (tablets)
  lg:  1024px (desktop)
  xl:  1280px (pantallas grandes)
-->

<!-- Navbar responsive -->
<nav class="flex items-center justify-between px-4 py-3">
  <span class="font-bold text-lg">⚡ EduCode</span>

  <!-- Visible solo en md+ -->
  <ul class="hidden md:flex gap-6 list-none">
    <li><a href="#">Módulos</a></li>
    <li><a href="#">Progreso</a></li>
  </ul>

  <!-- Botón hamburguesa: visible solo en móvil -->
  <button class="md:hidden text-2xl">☰</button>

  <!-- Botón login: siempre visible -->
  <a href="#" class="hidden md:block px-4 py-2 bg-cyan-500 rounded-xl text-sm">
    Ingresar
  </a>
</nav>

<!-- Texto responsive -->
<h1 class="text-2xl sm:text-3xl md:text-4xl lg:text-5xl font-bold">
  Aprende programación
</h1>

<!-- Grid responsive -->
<div class="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-4">
  <!-- tarjetas aquí -->
</div>
