01
Sintaxis básica y estructura
<?php ?> · echo · comentarios · include
▼

PHP es un lenguaje de scripting del lado servidor. Todo código PHP vive entre las etiquetas <?php y ?>. Las instrucciones terminan con punto y coma.

PHP
// Archivo: index.php
<?php

// Mostrar texto en pantalla
echo "Hola mundo";
echo "<br>";          // HTML dentro de PHP

// print es similar pero retorna 1
print("Hola de nuevo");

/* Comentario
   multilínea */

// Incluir otro archivo
include 'config.php';   // avisa si no existe
require 'funciones.php'; // detiene si no existe
include_once 'header.php';

?>
<!-- HTML fuera de PHP -->
<h1>Mi primera página PHP</h1>
💡
En archivos que son solo PHP (sin HTML) se omite el cierre ?> al final. Es una buena práctica que evita errores de espacios en blanco.
02
Variables y tipos de datos
$ · string · int · float · bool · null · var_dump
▼

Las variables en PHP empiezan siempre con $ y son de tipado dinámico (el tipo lo determina el valor asignado).

PHP
<?php

// Tipos de datos
$nombre   = "Carlos";      // string
$edad     = 25;             // integer
$precio   = 19.99;          // float
$activo   = true;           // boolean
$vacio    = null;           // null

// Verificar tipo
var_dump($edad);    // int(25)
gettype($precio);   // "double"

// Concatenación con punto (.)
$saludo = "Hola, " . $nombre . ". Tienes " . $edad . " años.";
echo $saludo;

// Interpolación dentro de comillas dobles
echo "Bienvenido, $nombre! Precio: $precio";

// Constantes (sin $)
define('VERSION', '1.0.0');
const SITIO = 'EduCode';
echo VERSION . ' - ' . SITIO;

03
Operadores
aritméticos · comparación · lógicos · ternario · null coalescing
▼
PHP
<?php

// Aritméticos
echo 10 + 3;   // 13
echo 10 - 3;   // 7
echo 10 * 3;   // 30
echo 10 / 3;   // 3.333...
echo 10 % 3;   // 1 (módulo/resto)
echo 2 ** 8;   // 256 (potencia)

// Comparación (¡doble == vs triple ===!)
$a = "5"; $b = 5;
$a == $b   // true  → compara valor (convierte tipo)
$a === $b  // false → compara valor Y tipo exacto
$a != $b   // false
$a !== $b  // true

// Lógicos
true && false  // false (AND)
true || false  // true  (OR)
!true          // false (NOT)

// Ternario: condición ? si_true : si_false
$edad = 20;
$msg = $edad >= 18 ? "Mayor de edad" : "Menor de edad";

// Null coalescing (??) — PHP 7+
$nombre = $_GET['nombre'] ?? "Invitado";
// Si $_GET['nombre'] no existe → "Invitado"

// Asignación compuesta
$total  = 100;
$total += 50;  // 150
$total -= 20;  // 130
$total *= 2;   // 260
⚠️
Usa siempre === (triple igual) para comparaciones en PHP. El == hace conversiones de tipo automáticas que generan bugs difíciles de rastrear: 0 == "hola" devuelve true.
04
Condicionales
if · elseif · else · switch · match
▼
PHP
<?php

$nota = 75;

// if / elseif / else
if ($nota >= 90) {
    echo "Excelente";
} elseif ($nota >= 70) {
    echo "Aprobado";
} elseif ($nota >= 60) {
    echo "Recuperación";
} else {
    echo "Reprobado";
}

// switch (usa == internamente)
$dia = "lunes";
switch ($dia) {
    case "lunes":
    case "martes":
        echo "Inicio de semana";
        break;
    case "viernes":
        echo "¡Fin de semana!";
        break;
    default:
        echo "Día normal";
}

// match — PHP 8+ (usa === y retorna valor)
$estado = "activo";
$color = match($estado) {
    "activo"   => "green",
    "inactivo" => "gray",
    "bloqueado" => "red",
    default    => "black",
};
echo $color; // green
✅
Prefiere match sobre switch en PHP 8+. Es más seguro (usa ===), más conciso y lanza excepción si ningún caso coincide (en vez de ignorarlo silenciosamente).
05
Bucles
while · do-while · for · foreach · break · continue
▼
PHP
<?php

// while — mientras se cumpla la condición
$i = 1;
while ($i <= 5) {
    echo "Vuelta $i<br>";
    $i++;
}

// do-while — ejecuta al menos una vez
$x = 10;
do {
    echo $x;
    $x--;
} while ($x > 0);

// for — cuando conoces el número de iteraciones
for ($j = 0; $j < 5; $j++) {
    echo "Elemento $j<br>";
}

// foreach — para recorrer arrays (el más usado en PHP)
$frutas = ["mango", "guayaba", "lulo"];
foreach ($frutas as $fruta) {
    echo $fruta . "<br>";
}

// foreach con clave => valor
$persona = ["nombre" => "Ana", "edad" => 28];
foreach ($persona as $clave => $valor) {
    echo "$clave: $valor<br>";
}

// break y continue
for ($n = 1; $n <= 10; $n++) {
    if ($n === 5) continue; // salta el 5
    if ($n === 8) break;    // para en 8
    echo $n . " ";           // 1 2 3 4 6 7
}
06
Funciones
function · parámetros · return · tipado · arrow functions
▼
PHP
<?php

// Función básica
function saludar($nombre) {
    return "Hola, $nombre!";
}
echo saludar("Carlos");

// Parámetros con valor por defecto
function calcularDescuento($precio, $descuento = 10) {
    return $precio - ($precio * $descuento / 100);
}
echo calcularDescuento(50000);        // 45000
echo calcularDescuento(50000, 20);   // 40000

// Tipado estricto — PHP 7+ (recomendado)
declare(strict_types=1);

function sumar(int $a, int $b): int {
    return $a + $b;
}

function obtenerNombre(): ?string { // ? = puede ser null
    return null;
}

// Funciones anónimas (closures)
$multiplicar = function($a, $b) {
    return $a * $b;
};
echo $multiplicar(4, 5); // 20

// Arrow functions — PHP 7.4+
$doble = fn($n) => $n * 2;
echo $doble(7); // 14

// Funciones útiles de string
strlen("hola");                    // 4
strtoupper("hola");               // HOLA
str_replace("a", "@", "hola");   // hol@
trim("  texto  ");               // "texto"
explode(",", "a,b,c");           // ["a","b","c"]
07
Arrays
indexados · asociativos · multidimensionales · array_map · array_filter
▼
PHP
<?php

// Array indexado
$colores = ["rojo", "verde", "azul"];
echo $colores[0];   // rojo
echo count($colores); // 3

// Array asociativo (clave → valor)
$estudiante = [
    "nombre"   => "Laura",
    "ficha"    => "2876543",
    "programa" => "Análisis de Sistemas",
    "nota"     => 4.5,
];
echo $estudiante["nombre"]; // Laura

// Array multidimensional
$cursos = [
    ["nombre" => "Bases de Datos", "horas" => 40],
    ["nombre" => "PHP",             "horas" => 36],
    ["nombre" => "Laravel",         "horas" => 48],
];
foreach ($cursos as $curso) {
    echo $curso["nombre"] . ": " . $curso["horas"] . "h<br>";
}

// Funciones esenciales de arrays
array_push($colores, "amarillo");     // agrega al final
array_pop($colores);                   // elimina el último
in_array("rojo", $colores);           // true/false
array_keys($estudiante);              // ["nombre","ficha"...]
array_values($estudiante);            // ["Laura","2876543"...]
sort($colores);                        // ordena indexado
ksort($estudiante);                   // ordena por clave

// array_map — transforma cada elemento
$notas  = [3.5, 4.0, 2.8, 4.9];
$dobles = array_map(fn($n) => $n * 2, $notas);

// array_filter — filtra elementos
$aprobados = array_filter($notas, fn($n) => $n >= 3.0);

// array_reduce — reduce a un valor
$total = array_reduce($notas, fn($carry, $item) => $carry + $item, 0);
$promedio = $total / count($notas); // 3.8
08
Formularios y superglobales
$_GET · $_POST · $_SESSION · validación · sanitización
▼
PHP
<!-- formulario.html -->
<form action="procesar.php" method="POST">
    <input type="text"  name="nombre" required>
    <input type="email" name="correo" required>
    <input type="number" name="edad">
    <button type="submit">Enviar</button>
</form>
PHP — procesar.php
<?php

// Solo procesar si es POST
if ($_SERVER['REQUEST_METHOD'] === 'POST') {

    // 1. Recibir datos
    $nombre = $_POST['nombre'] ?? '';
    $correo = $_POST['correo'] ?? '';
    $edad   = $_POST['edad']   ?? 0;

    // 2. Sanitizar (limpiar)
    $nombre = htmlspecialchars(trim($nombre));
    $correo = filter_var($correo, FILTER_SANITIZE_EMAIL);
    $edad   = (int) $edad;

    // 3. Validar
    $errores = [];
    if (empty($nombre)) {
        $errores[] = "El nombre es obligatorio";
    }
    if (!filter_var($correo, FILTER_VALIDATE_EMAIL)) {
        $errores[] = "El correo no es válido";
    }
    if ($edad < 15 || $edad > 80) {
        $errores[] = "Edad fuera de rango";
    }

    // 4. Procesar o mostrar errores
    if (empty($errores)) {
        echo "Registro exitoso para $nombre";
    } else {
        foreach ($errores as $error) {
            echo "<p style='color:red'>$error</p>";
        }
    }
}

// $_GET — datos en la URL (?id=5)
$id = filter_input(INPUT_GET, 'id', FILTER_VALIDATE_INT);

// $_SESSION
session_start();
$_SESSION['usuario'] = $nombre;
echo $_SESSION['usuario'];
session_destroy(); // cerrar sesión
⚠️
Nunca uses datos de $_POST o $_GET directamente en SQL o HTML. Siempre sanitiza primero con htmlspecialchars() y luego valida antes de procesar.
09
Programación Orientada a Objetos (OOP)
class · constructor · herencia · interfaces · visibilidad
▼
PHP — OOP completo
<?php

// Clase base
class Persona {
    // Propiedades con visibilidad
    public    string $nombre;
    protected int    $edad;
    private   string $dni;

    // Constructor
    public function __construct(
        string $nombre,
        int    $edad,
        string $dni
    ) {
        $this->nombre = $nombre;
        $this->edad   = $edad;
        $this->dni    = $dni;
    }

    public function presentarse(): string {
        return "Soy {$this->nombre}, tengo {$this->edad} años.";
    }

    // Getter para propiedad privada
    public function getDni(): string {
        return $this->dni;
    }
}

// Herencia
class Estudiante extends Persona {
    public string $ficha;
    public string $programa;

    public function __construct(
        string $nombre, int $edad, string $dni,
        string $ficha,  string $programa
    ) {
        parent::__construct($nombre, $edad, $dni);
        $this->ficha   = $ficha;
        $this->programa = $programa;
    }

    // Sobreescribir método
    public function presentarse(): string {
        return parent::presentarse() . " Ficha: {$this->ficha}";
    }
}

// Interface
interface Calificable {
    public function calcularPromedio(): float;
    public function aprobo(): bool;
}

// Clase que implementa interface
class EstudianteConNotas extends Estudiante implements Calificable {
    private array $notas;

    public function calcularPromedio(): float {
        return array_sum($this->notas) / count($this->notas);
    }

    public function aprobo(): bool {
        return $this->calcularPromedio() >= 3.0;
    }
}

// Uso
$est = new Estudiante("Laura", 22, "12345678", "2876543", "Análisis");
echo $est->presentarse();
✅
La OOP en PHP es la base directa de Laravel. Todo lo que aprendas aquí (clases, herencia, interfaces, visibilidad) lo verás nuevamente en Eloquent Models, Controllers y Service Providers.
