Variables y tipos de datos
$var · string · int · float · bool · null · var_dump
▼

En PHP todas las variables empiezan con $. PHP es de tipado dinámico: la variable toma el tipo del valor que le asignas.

📁
Crea una carpeta practica_php/ en tu escritorio. Dentro crea 01_variables.php y ábrelo en PhpStorm. Usa WAMP para ejecutarlo: http://localhost/practica_php/01_variables.php
PHP
<?php

// Tipos básicos
$nombre     = 'EduCode CEFIT';    // string
$modulos    = 5;                  // int
$precio     = 0.0;                // float (gratis!)
$activo     = true;               // bool
$descripcion = null;              // null = sin valor

// Ver el tipo y valor de una variable
var_dump($nombre);     // string(13) "EduCode CEFIT"
var_dump($modulos);    // int(5)
var_dump($activo);     // bool(true)

// Concatenación con punto
echo "La plataforma " . $nombre . " tiene " . $modulos . " módulos.";

// Interpolación dentro de comillas dobles
echo "<br>Plataforma: {$nombre}, módulos: {$modulos}";

02
Control de flujo
if · else · elseif · for · foreach · while · match
▼

El control de flujo permite ejecutar código condicionalmente o repetirlo. Es la base de toda lógica de programación.

PHP
<?php

$progreso = 75; // % de lecciones completadas

// if / elseif / else
if ($progreso === 100) {
    echo "🎉 ¡Módulo completado!";
} elseif ($progreso >= 50) {
    echo "📚 Vas por buen camino: {$progreso}%";
} else {
    echo "🚀 Empieza a estudiar ya!";
}

// foreach — iterar un array
$modulos = ['Bases de Datos', 'PHP', 'HTML', 'Laravel', 'Moonshine'];

foreach ($modulos as $indice => $nombre) {
    echo "<br>" . ($indice + 1) . ". " . $nombre;
}

// match — alternativa moderna al switch
$nivel = 'intermedio';
$emoji = match($nivel) {
    'basico'     => '🟢',
    'intermedio' => '🟡',
    'avanzado'   => '🔴',
    default      => '⚪',
};
echo "<br>Nivel: {$emoji} {$nivel}";

03
Funciones
function · parámetros · return · tipos · arrow functions
▼

Las funciones encapsulan lógica reutilizable. En PHP moderno (8+) podemos declarar los tipos de parámetros y retorno, lo que hace el código más seguro y claro.

PHP
<?php

// Función con tipos declarados (PHP 8+)
function calcularProgreso(int $completadas, int $total): float
{
    if ($total === 0) return 0.0;
    return round(($completadas / $total) * 100, 1);
}

echo calcularProgreso(3, 5); // 60.0

// Parámetros con valor por defecto
function saludarEstudiante(string $nombre, string $modulo = 'General'): string
{
    return "Hola {$nombre}, estás en el módulo: {$modulo}";
}

echo saludarEstudiante('Ana');
echo saludarEstudiante('Luis', 'PHP Puro');

// Arrow function (función flecha, PHP 7.4+)
$estaActivo = fn(bool $activo): string => $activo ? '✅ Activo' : '❌ Inactivo';

echo $estaActivo(true);  // ✅ Activo
echo $estaActivo(false); // ❌ Inactivo

04
Arrays
array indexado · asociativo · multidimensional · array_* functions
▼

Los arrays son la estructura de datos más usada en PHP. En un LMS los usaremos para listas de módulos, lecciones, ejercicios y datos del usuario.

PHP
<?php

// Array indexado
$nombres = ['Bases de Datos', 'PHP', 'HTML', 'Laravel', 'Moonshine'];
echo count($nombres); // 5

// Array asociativo (clave => valor)
$modulo = [
    'nombre'     => 'PHP Puro',
    'icono'      => '🐘',
    'orden'      => 2,
    'activo'     => true,
    'lecciones'  => 6,
];

echo $modulo['nombre'];  // PHP Puro

// Array multidimensional (como resultado de una BD)
$modulos = [
    ['id' => 1, 'nombre' => 'Bases de Datos', 'lecciones' => 4],
    ['id' => 2, 'nombre' => 'PHP Puro',       'lecciones' => 6],
    ['id' => 3, 'nombre' => 'HTML & CSS',     'lecciones' => 5],
];

foreach ($modulos as $m) {
    echo "<br>#{$m['id']} {$m['nombre']}: {$m['lecciones']} lecciones";
}

// Funciones útiles de array
$total = array_sum(array_column($modulos, 'lecciones'));
echo "<br>Total lecciones: {$total}"; // 15

05
Clases y Objetos (OOP)
class · __construct · properties · methods · visibility
▼

La Programación Orientada a Objetos (POO) organiza el código en clases que modelan entidades del mundo real. Laravel está 100% construido con OOP.

PHP
<?php

class Modulo
{
    // Propiedades
    public int $id;
    public string $nombre;
    public string $icono;
    private bool $activo;
    private array $lecciones = [];

    // Constructor
    public function __construct(int $id, string $nombre, string $icono)
    {
        $this->id     = $id;
        $this->nombre = $nombre;
        $this->icono  = $icono;
        $this->activo = true;
    }

    // Métodos
    public function agregarLeccion(string $titulo): void
    {
        $this->lecciones[] = $titulo;
    }

    public function totalLecciones(): int
    {
        return count($this->lecciones);
    }

    public function resumen(): string
    {
        $estado = $this->activo ? 'activo' : 'inactivo';
        return "{$this->icono} {$this->nombre} — {$this->totalLecciones()} lecciones ({$estado})";
    }
}

// Crear instancias
$bd  = new Modulo(1, 'Bases de Datos', '🗄️');
$php = new Modulo(2, 'PHP Puro', '🐘');

$bd->agregarLeccion('Crear tablas');
$bd->agregarLeccion('Claves foráneas');

echo $bd->resumen();   // 🗄️ Bases de Datos — 2 lecciones (activo)
echo $php->resumen();  // 🐘 PHP Puro — 0 lecciones (activo)

06
Herencia e Interfaces
extends · parent:: · abstract · interface · implements
▼

La herencia permite crear clases más específicas basadas en otras. Laravel usa extensivamente este patrón (todos los modelos extienden Model, todos los controladores extienden Controller).

PHP
<?php

// Clase base (padre)
abstract class Contenido
{
    public function __construct(
        public readonly int    $id,
        public readonly string $titulo,
        public readonly int    $orden
    ) {}

    // Método abstracto: DEBE implementarse en la clase hija
    abstract public function tipo(): string;

    // Método concreto: se hereda tal cual
    public function info(): string
    {
        return "[{$this->tipo()}] #{$this->orden}: {$this->titulo}";
    }
}

// Clase hija
class Paso extends Contenido
{
    public function __construct(
        int    $id,
        string $titulo,
        int    $orden,
        public readonly string $contenido
    ) {
        parent::__construct($id, $titulo, $orden);
    }

    public function tipo(): string { return 'PASO'; }
}

$paso = new Paso(1, 'Instalar MySQL', 1, 'Descarga WAMP...');
echo $paso->info(); // [PASO] #1: Instalar MySQL

Formularios HTML + PHP
$_POST · $_GET · filter_input · htmlspecialchars · validación
▼

Un formulario HTML envía datos al servidor. PHP los recibe en $_POST o $_GET. Siempre debemos validar y sanitizar antes de usar esos datos.

🔒
Nunca insertes $_POST directamente en HTML o SQL. Usa htmlspecialchars() para mostrar y prepared statements para BD. Verás esto en acción en el paso de PDO.
PHP
<?php
// formulario.php — misma página para mostrar y procesar

$mensaje = '';

if ($_SERVER['REQUEST_METHOD'] === 'POST') {

    // Obtener y sanitizar datos
    $nombre = trim($_POST['nombre'] ?? '');
    $correo = filter_input(INPUT_POST, 'correo', FILTER_VALIDATE_EMAIL);

    // Validar
    if (empty($nombre)) {
        $mensaje = '❌ El nombre es obligatorio.';
    } elseif (!$correo) {
        $mensaje = '❌ Correo inválido.';
    } else {
        $nombreSafe = htmlspecialchars($nombre, ENT_QUOTES, 'UTF-8');
        $mensaje = "✅ Bienvenido, {$nombreSafe}! Correo: {$correo}";
    }
}
?>
<!DOCTYPE html>
<html><body>
<form method="POST">
  <input type="text"  name="nombre" placeholder="Tu nombre">
  <input type="email" name="correo" placeholder="tu@correo.com">
  <button type="submit">Enviar</button>
</form>
<?php if($mensaje) echo "<p>{$mensaje}</p>"; ?>
</body></html>

08
PDO — PHP conecta con MySQL
PDO · prepare · execute · fetchAll · prepared statements
▼

PDO (PHP Data Objects) es la forma segura y moderna de conectar PHP con bases de datos. Usa prepared statements que evitan SQL Injection.

🔐
Los prepared statements separan el SQL del dato. El motor de BD procesa la estructura del query primero y los datos después — imposible de inyectar código SQL malicioso.
PHP
<?php

// Conexión a la BD (igual que el .env de Laravel)
$dsn  = 'mysql:host=127.0.0.1;dbname=programando;charset=utf8mb4';
$user = 'root';
$pass = 'base1234';

try {
    $pdo = new PDO($dsn, $user, $pass, [
        PDO::ATTR_ERRMODE            => PDO::ERRMODE_EXCEPTION,
        PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
    ]);

    // Leer todos los módulos activos
    $stmt = $pdo->query('SELECT id, nombre, icono FROM modulos WHERE activo = 1 ORDER BY orden');
    $modulos = $stmt->fetchAll();

    foreach ($modulos as $m) {
        echo "{$m['icono']} {$m['nombre']}<br>";
    }

    // Buscar lecciones de un módulo (prepared statement)
    $moduloId = 1; // viene de $_GET en la realidad
    $stmt = $pdo->prepare('SELECT titulo FROM lecciones WHERE modulo_id = ? AND activo = 1');
    $stmt->execute([$moduloId]);
    $lecciones = $stmt->fetchAll();

    echo "<br>Lecciones del módulo 1:<br>";
    foreach ($lecciones as $l) {
        echo "— {$l['titulo']}<br>";
    }

} catch (PDOException $e) {
    echo 'Error de conexión: ' . $e->getMessage();
}
